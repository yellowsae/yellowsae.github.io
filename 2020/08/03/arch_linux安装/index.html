arch_linux安装
发表于 2020-08-03| 更新于 2020-08-04
字数总计:712|阅读时长: 3 分钟
| 阅读量:

开始

准备 ：

参考官方文档 https://wiki.archlinux.org/index.php/Installation_guide_

下载archLinux镜像 : https://www.archlinux.org/download/ (也可以去清华园，中科大下载开源镜像）

环境： VMware （ 也可将arch_linux 镜像做成一个启动盘，然后安装电脑）

键盘布局 ： 无

1， ip link ： 查看网络状态

如果是连接WiFi : ip link set wlan0 up : 开启WiFi

iwlist wlan0 scan | grep ESSID : 扫描WiFi

wpa_passphrase > internet.conf : 配置文件

wpa_supplicant -c internet.conf -i· wlan0 & : 连接WiFi

dhcpcd & : 动态分配IP

然后ping baidu.com

2， 更新系统时间 ： timedatectl set-ntp true ： 设置时间

3， 建立磁盘分区

查看分区 ： fdisk -l

进入分区 : fdisk /dev/sda

p : 查看当前磁盘的分区

g : 新建一个 GPT 磁盘

n ： 创建分区

1， boot 分区 : 系统分区

开始0 (空格)

到+512M

3， SWPA ： 虚拟内存分区 ， 不用特别大

开始 0（空格）

到 +1G

2， 主分区 ：

开始0 (空格)

到 尾 （空格）

w : 写入磁盘 （最后一步 确定好 再 写入磁盘 ）

4， 修改分区格式

将系统磁盘变成 fat 格式 : mkfs.fat -F32 /dev/sda1

将系统 主分区 变成 ext4 格式 ： mkfs.ext4 /dev/sda2

将SWAP分区变成 SWAP格式 ： mkswap /dev/sda3

​ swapon /dev/sda3

5， 挂载分区

将储存空间挂载到 主分区下 : mount /dev/sda2 /mnt

将系统盘挂载在/mnt/boot 下 ： mkdir /mnt/boot

​ mount /dev/sda1 /mnt/boot

6 ， 切换国内源

​ vim /etc/pacman.d/mirror

​ 把China 源放在第一位

7， 安装Linux内核 和框架 ： pacstrap /mnt base linux linux-firmware

8， 生成fstab 文件 ： genfst ab -U /mnt >> /mnt/etc/fstab : 生成fstab 文件

进入系统 : arch-chroot /mnt

设置时区 ： ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

hwclock –systohc 同步

安装 必须工具 ： pacman -S vim zsh wpa_supplicant dhcpcd iw dialog intel-ucode grub efibootmgr

< 进入系统后安装>

再执行 ： grub-install –target=x86_64-efi –efi-directory=/boot –bootloader-id=grub

执行 grub-mkconfig -o /boot/grub/grub.cfg

设置密码 ： passwd

exit

umount -R /mnt

reboot

systemctl enable dhcpcd

systemctl start dhcpcd

pacman -S xorg xorg-xinit

pacman -S xorg-xrandr

pacman -S ttf-dejavu wqy-microhei <字体包>

useradd -m -g users -G wheel -s /bin/bash yellosea // 设置用户

passwd yellowsea // 设置用户密码

pacman -S sudo

pacman -S vi

visudo

去掉 %whell 前的# 号

pacman -S xfce4

pacman -S xfce4-goodies

pacman -S lightdm

pacman -S lightdm-gtk-greeter

systemctl enable lightdm

pacman -S xdg-user-dirs

reboot

sudo vim /etc/locale.gen

去掉 en.US.UTF-8

zh.CN.UTF-8

的# 号

设置语言 : vim /mnt/etc/locale.conf

添加 LANG=en_US.UTF-8

LANG=zh_CN.UTF-8

sudo locale-gen

sudo vim /etc/pacman.conf

sudo pacman -Syy

sudo pacman -S yay yaourt

sudo pacman -Syu

sudo pacman -Syy

sudo pacman -S google-chrome

pacman -S netease-cloud-music <网易云>

pacman -S pulseaudio pulseaudio-alsa alsa-utils alsa-oss

pacman -S fcitx-im fcitx-configtool fcitx-sogoupinyin

pacman -S pavucontrol

pacman -S ntfs-3g

pacman -S gvfs-mtp

编译hostname : vim /mnt/etc/hostname

添加 名字 ： Yellowsea

编译host 文件 ： vim /mnt/etc/hosts

添加 ：

127.0.0.1 localhost

::1 localhost

127.0.0.1 Yellowsea.localdomain Yellowsea

reboot

sudo pacman -S screenfetch

screenfetch
