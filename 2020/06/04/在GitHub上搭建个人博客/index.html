在GitHub上搭建个人博客
发表于 2020-06-04| 更新于 2020-06-04
字数总计:106|阅读时长: 1 分钟
| 阅读量:

搭建步骤：

1, 创建一个名为 “用户名.github.io” 的仓库

2, 创建好后要在仓库中创建一个名为 index.html 文件， 并且要删除 README.md文件

3, 访问 Project Pages 项目站点

4, 在仓库里的点击settings， 找到[launch automatic page generator] 来自动生成主题页面

5， 选择你喜欢的主题，并生成网页

5, 我的搭建效果 ： 我的博客
